# nonBlockingTask.js — 이벤트 루프 논블로킹

## 개요
무거운 반복 연산을 한 번에 돌리지 않고 `setTimeout`으로 잘게 쪼개, 여러 이벤트 루프 틱에 나눠 처리하는 예제.

## 핵심 개념
- 큰 반복문을 한 번에 실행하면 그 시간 동안 **메인 스레드가 막혀** UI/다른 작업이 멈춘다(블로킹).
- 작업을 작은 단위로 나누고 `setTimeout(..., 0)`으로 다음 틱에 넘기면, 작업 **사이사이에 다른 코드가 끼어들 수 있어** 블로킹을 막는다.

## 코드 설명
```js
let i = 0;
function lightTask() {
    if (i >= 1_000_000_000_000) return; // 종료 조건

    for (let j = 0; j < 1000_000; j++) i++; // 한 틱에 100만 번만 처리

    setTimeout(lightTask, 0); // 남은 작업을 다음 틱으로 넘김
}
lightTask();
```

## 실행
```bash
node async/nonBlockingTask.js
```
> 이 예제는 `console.log`가 없어 실행해도 겉으로는 조용히 끝난다. 동작을 확인하려면 `lightTask` 안에 진행 로그를 추가하는 것을 권장한다.

## 주의
- 목표치가 1조(`1_000_000_000_000`)라 끝까지 돌리면 매우 오래 걸린다. 학습·확인용으로는 10억 정도로 줄이는 것이 좋다.
- 중간에 멈추려면 `Ctrl + C`.
